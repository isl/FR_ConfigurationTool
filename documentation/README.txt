This folder contains the existing documentation regarding this project.

The files contained are: 

1)'Fundamental Categories and Relationships for Intuitive querying CIDOC-CRM based repositories'.docx:
   --------------------------------------------------------------------------------------------------
   This is the latest version of the Technical Report 
  'Tzompanaki, K., & Doerr, M. (2012). Fundamental Categories and Relationships for Intuitive querying CIDOC-CRM based repositories'
   which exists ubder the link: http://www.ics.forth.gr/tech-reports/2012/2012.TR429_Intuitive_querying_CIDOC-CRM.pdf .
   It describes what each FR stands for and provides the respective path in the CIDOC-CRM model.

2) 'Master_Aikaterini_Tzompanaki.pdf':
   ---------------------------------
   This is the master thesis conducted upon the Fundamental Categories and Fundamental Relationships model, with title 
   'Design and Implementation of a tool for formulating recall-oriented structured queries on semantic networks'.
   It is written by Aikaterini Tzompanaki, under the supervision of Mr.Martin Doerr and Mr.Yannis Tzitzikas

 	 
3) 'MasterThesisPresentation_Tzompanaki.pptx':
   ----------------------------------------- 
   This is the presentation for the master thesis 'Design and Implementation of a tool for formulating recall-oriented structured queries on semantic networks' defence, written by Aikaterini Tzompanaki.	

4) 'SubRelationships.docx': 
   -----------------------------------------    
   This document contains for selected FRs, the defined sub-relationships i a list format.

5) folder 'javadoc':
   ----------------
   Contains the javadoc for the 'Fundamental Configuration Tool'  source code.
